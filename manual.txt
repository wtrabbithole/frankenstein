1- first_run для полной очистки git
2- vromfs_unpacke.exe распаковка aces и  char
3- full_unpack_aces ( full_unpack_aces.bat >> aces.log)
4- full_unpack_char (full_unpack_char.bat >> char.log)
5- git_add_and_commit.cmd 
6 в консоли- git log

first_run.bat- запускать только самый первый раз, или если надо сбросить всю историю

vromfs_unpack.cmd- запускать после добавления aces и char файлов
скрипт сначала удаляет все старое(папки, логи), потом распаковывает новое

full_unpack_aces.bat- распаковывает все файлы в папке aces
запускать через команду full_unpack_aces.bat >> aces.log
она создает лог работы и можно будет посмотреть, какие файлы не распаковались

full_unpack_char.bat- то же самое для второго файлы. запускать full_unpack_char.bat >> char.log

git_add_and_commit.cmd-добавляет обе папки с распакованными файлами, оба бинарных файла и оба лога в гит, читает номер версии и коммитит с этим названием
